# 时间切片 (Time Slicing)



两种交互类型，被动（动画，弹窗广告，轮播），被动（鼠标，键盘）



FPS是图像领域中的定义，是指画面每秒传输帧数，通俗来讲就是指动画或视频的画面数。
FPS”也可以理解为我们常说的“刷新率（单位为Hz）”



60HZ = 每秒刷新60次
(1 秒 = 1000 毫秒) / 60 帧 = 16.7 毫秒/帧



让用户感觉到流畅

响应时间（主动交互）：100ms内
动画：16.7ms内   
空闲TODO : 50ms内
加载时间：1000ms内


js+style+layout+paint+composite  总时间在16.7ms内
